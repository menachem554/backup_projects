web server and client: ==                                                   בנמשל הבית זה הקלינט הצורך את האינטרנט. והחנות זה השרת שמספק את המידע 

Your internet connection: ==                                 חיבור האינטרנט זה מה שמאפשר ללקוח לבקש ולקבל את המידע כמו הרחוב שבין החנות והבית

TCP/IP ==                                                 פרוטוקול בקרת שידור ופרוטוקול אינטרנט הם הפרוטוקלים המגדירים איך המידע יעבור ברשת האינטרנט

DNS == שמות השרתים הם כמו ספר טלפונים לאתרי אינטרנט שהאתר בעצם מוצא על איזה שרת האתר שלך נמצא כדי שהוא יכול לשלוח אליו בקשות

HTTP == פרוטוקול העברת היפרטקסט משמש כדי להעביר מידע שגם הקלינט וגם השרת יוכלו להבין

       
                                                                     מה קורה כאשר אני מקליד כתובת אתר בדפדפן שלי
                                                                     -------------------------------------------------------

1) הדפדפן שלך מחפש את הכתובת של השרת שבו מאוחסן האתר שביקשת 
2) הדפדפן שולח בקשה לשרת בשפת ההיפרטקסט שהם מבינים באמצעות פרוטוקול התקשורת ומבקש עותק של האתר בשביל הקלינט
3)אם השרת מאשר את הבקשה של הדפדפן הוא שולח אישור 200 ושולח את הקבצים שהדפדפן צריך כדי להציג את המידע בחבילות מידע
4)הדפדפן מחבר את כול חבילות המידע לעמוד אינטרנט ומציג אותו לקלינט
